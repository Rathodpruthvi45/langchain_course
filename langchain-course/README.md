# LangChain Course

📚 **Complete LangChain course** - from basics to advanced with interactive modules and real-world examples.

## Getting Started

```bash
git clone https://github.com/yourusername/langchain-course.git
cd langchain-course
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Features

- 🎯 Interactive module system with progress tracking
- 📝 Code examples with real-world patterns
- 💻 Complete LangChain curriculum
- 🎨 Dark theme design
- ⚡ Built with React + Vite
